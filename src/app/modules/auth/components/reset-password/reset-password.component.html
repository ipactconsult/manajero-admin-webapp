Best practices for creating a strong password

  • Use a combination of letters (upper and lower case), numbers and special characters
  • Use a unique password for each account
  • Avoid using personal or easy-to-guess information
  • Use a long and complex password
  • Avoid using common words or sequences of letters or numbers (like "123456" or "password")

Take the time to create strong and secure passwords.
